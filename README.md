# CCC Premium Dark Deploy Package

This version is styled to match the premium dark canvas direction more closely.

## Included
- `index.html`
- `styles.css`
- `app.js`

## Deploy
Upload these files to:
- Vercel
- Netlify
- GitHub Pages
- any static host

## iPhone use
After deployment:
1. Open the site in Safari
2. Tap Share
3. Tap Add to Home Screen

## Notes
- Saves check-ins to local storage on the current device
- Front-end beta only
- Next version can add login, cloud sync, charts, photos, reminders, and real branding assets
